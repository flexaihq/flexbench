# FlexBoard

## Installation

```bash
# Go to the flexboard directory
cd flexboard

# Create a virtual environment
python -m venv .venv

# Activate the virtual environment
source .venv/bin/activate

# Install the required packages
pip install -r requirements.txt

# Run the application
python -m app
```
